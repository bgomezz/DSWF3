Selectores en CSS


Los selectores se usan para darle un estilo especifico a cada selector, en el lenguaje de CSS, toda línea es una regla, por ejemplo, strong, que es un selector, permite a que elementos en el DOM se le aplica esta regla.

Con respecto al código anterior, la parte interior de las llaves es la declaración, la palabra clave es color, que es una propiedad, el rojo es el valor y el punto y coma despúes de definir el par propiedad-valor, separar los posibles otros pares propiedad-valor en una misma declaración.

Como se puede dar cuenta se da valores los atributos de los selectores. Esto permite que sus reglas sean más específicas.

Dos atributos tienen un estatus especial para CSS. Son class e id.

Class: Selectores de clase

El atributo class se utiliza para asignarle a un elemento un nombre de clase (class). Depende del nombre que elijas como class. Múltiples elementos en un documento pueden tener el mismo valor de la clase.
En su hoja de estilo, escriba un punto (.) antes del nombre de la clase, cuando desee utilizar un selector.

Selectores ID

Utilice el atributo id en un elemento para asignar una identificación para ese elemento. Depende del nombre que elija para el ID. El nombre de ID debe ser único en el documento.
En su hoja de estilo, escriba el símbolo de almohadilla (#) antes del ID cuando se utiliza esto en un selector.
Ejemplo
El siguiente ejemplo usa una etiqueta HTML que tiene un atributo class y un atributo id:

<p class="key" id="principal">

Si más de una regla se aplica a un elemento y especifica la misma propiedad, entonces el CSS da prioridad a la regla que tiene el selector más específico. Un selector ID es más específico que un selector de class, que a su vez es más específico que un selector (tag) de etiquetas.

Cuando tenga un problema con las reglas en conflicto, trate de resolverlo haciendo una de las normas más específica, para que tenga prioridad. Si no puede hacerlo, entonces trate de mover una de las reglas más cerca del final de la hoja de estilos, para que tenga prioridad.

Las pseudo-clases de los selectores
Una pseudo-clase o pseudo-class es en CSS una palabra clave añadida a los selectores, que epecifica un estado especial del elemento a seleccionar. Por ejemplo {{Css Href (": hover")}} aplicará un estilo cuando el usuario se desplaza sobre el elemento especificado mediante el selector.

Las Pseudo-classes, junto con los pseudo-elements, permiten un estilo a un elemento no sólo en relación con el contenido de la estructura del documento, sino también en relación a los factores externos, como la historia del navegador, por ejemplo: ({{href css (": visited ")}},), el estado de su contenido como {{ref css (": :checked")}} en algunos elementos del formulario, o la posición del ratón {{cssxref (":hover ")}}, que le permite saber si el ratón está sobre un elemento o para ver una lista completa de selectores, visite especificación CSS3 permite saber si el ratón está sobre un elemento o no.
