Etiqueta display de CSS3


CSS Layout - display: inline-block

El display: inline-block Value En comparación con display: inline, la principal diferencia es que display: inline-block permite establecer un ancho y alto en el elemento.

Además, con display: inline-block, se respetan los márgenes / rellenos superiores e inferiores, pero con display: inline no se logra esto.

En comparación con display: block, la principal diferencia es que display: inline-block no agrega un salto de línea después del elemento, por lo que el elemento puede agregarse junto a otros elementos.

El siguiente ejemplo muestra el diferente comportamiento de display: inline, display: inline-block y display: block:

Ejemplo:

span.a {
display: inline; /* the default for span */
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}

span.b {
display: inline-block;
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}

span.c {
display: block;
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}
 Usar inline-block para crear enlaces de navegación

Un uso común para display: inline-block es mostrar elementos de la lista horizontalmente en lugar de verticalmente. El siguiente ejemplo crea enlaces de navegación horizontal:
Ejemplo
.nav {
background-color: yellow;
list-style-type: none;
text-align: center;
padding: 0;
margin: 0;
}

.nav li {
display: inline-block;
font-size: 20px;
padding: 20px;

}
