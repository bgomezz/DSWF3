Listas HTML


Listas no ordenadas: <ul>

Estas listas no ordenadas se construyen con la etiqueta HTML <ul> cerrando respectivamente la etiqueta con </ul> se usa para organizar listas con puntos, así que para cada punto que desee añadir a la lista, debe colocarlo entre la etiqueta <li>. Es decir, <li> Objeto de la lista</li> de esta manera se crea la lista con un punto negro, pero si deseas utilizar otros símbolos, lo puedes hacer a través del atributo “type”.

Para usar el atributo de la etiqueta <li> se hace de la siguiente manera:
<ul>
<li type= “disc”> dibuja un punto un punto negro </li>
<li type= “circle”> dibuja un punto sin relleno y contorno negro. </li>
<li type= “square”> dibuja un cuadro. </li>
</ul>

Listas ordenadas: <ol>

Para enumerar listas de manera organizada, se utiliza la etiqueta HTML <ol>, se coloca cada elemento de la lista entre las etiquetas <li>Objeto de la lista</li> en la medida en que vaya agregando cada uno de los ítems que desea organizar, esta etiqueta los va a enumerar automáticamente, pero si quiere definir a partir de qué número debe empezar a organizar, se usa la propiedad value, se realiza de la siguiente manera:
<ol>
<li value= “8”> Primer ítem que enumera iniciando en 8 </li>
<li> Enumera el siguiente ítem automáticamente, sería 9 </li>
<li> Luego sigue el ítem número 10. Y Así sucesivamente </li>
</ol>

Listas de definiciones: <dl>, <dt> y <dd>
Se utilizan para hacer un listado de definiciones, la etiqueta dl viene de los términos en inglés “Definiton list”, que indica dentro de ella, en ella y su cierre, va a ir una definición.
En el caso de la etiqueta dt viene de los términos “Definition term”, dentro de ella irá el término que vamos a definir, es decir, dentro de <dt> se escribe el título de la definición.
Y la etiqueta <dd> viene de los términos “Definition description” dentro de ella irá la definición. Si escribe varios listados de definición, éstas se separarán automáticamente entre ellas para facilitar su diferenciación. Aquí podemos ver un ejemplo de código de dos listados de definición:

<dl>
<dt>Aquí va el término a definir</dt>
<dd>Aquí defines el término</dd>
</dl>
<dl>
<dt>El siguiente término que se va a definir</dt>
<dd>Aquí va la segunda definición, separada automáticamente de la anterior</dd>
</dl>

El elemento de sección de navegación <nav>
El elemento <nav> HTML se utiliza para representar secciones de las páginas, proporcionando enlaces de navegación dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices.
No es necesario que todos los enlaces estén contenidos en un elemento <nav>contenido</nav> está destinada sólo para el bloque principal de enlaces de navegación; por ejemplo, el elemento <footer> a menudo tiene una lista de enlaces que no necesitan estar en un elemento <nav>.

Ejemplo
En este ejemplo un <nav> bloque se usa para contener una lista desordenada ( <ul>) de enlaces. Con el CSS apropiado, esto se puede presentar como una barra lateral, barra de navegación o menú desplegable.
<nav class="menu">
<ul>
<li><a href="#">Inicio</a></li>
<li><a href="#">Lecturas</a></li>
<li><a href="#">Multimedia</a></li>
</ul>
</nav>
